# @turnkey/stacks

## 0.1.0-beta.1

### Minor Changes

- Initial beta release of `@turnkey/stacks`
- Added `TurnkeySigner` class for signing Stacks transactions with Turnkey
- Added `signSTXTransfer` method for signing STX token transfers
- Added `broadcastTransaction` helper function
- Added `signAndBroadcastSTXTransfer` convenience function
- Added `getAddressFromPublicKey` utility function
- Support for both testnet and mainnet networks
- Automatic nonce fetching from Hiro API
